发表于 2020-03-31 更新于 2020-04-01 分类于 hexo入门

简介

简单了解hexo个人静态博客的使用…

常用命令

hexo g重新编辑本地博客

hexo s本地运行博客（默认端口运行）

  • hexo s -p 5000 端口：5000的本地运行
  • Ctrl + C结束本地运行（Y）

hexo d上传到github上

编写文章

hexo n '文章名称'创建文章

首页文章截取

安装图片插件

在_config.yml配置文件中，修改为 post_asset_folder: true， 然后新建一篇文章，然后会生成xx.md及xx文件夹‘用于存放图片’
